PacMen Exercise Repository

An exercise about images movement.
A pacman image moves its mouth as eating points.
Furthermore, the image moves horizontally across the screen.
When the images gets to touch the edge of the screen, it must reverse and move to the opposite edge.
For this exercise, we use 4 pacmen images.

Eye Exercise Repository

An exercise about mouse position recognition.
First we create oval shapes to generate an eye "image".
Then, we have to recognize the mouse position in order to change the direction where the eye looks.
Finally, we have to create a second eye and apply the same logic from the first eye.

Real Time Bus Tracking Repository

An exercise about bus stops tracking.
First we make use of the mapbox API.
Then we show the bus stops in the map.
